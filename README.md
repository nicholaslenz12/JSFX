# JSFX
A collection of audio effects written as JSFX. There are compatible with REAPER
(Digital Audio Workstation).

## The current effects are:
- N-Compressor
- N-Delay
- N-Distortion
- N-Sidechainer
- N-Tremolo
- Bitcrusher
- Filter
- Mono
